# Frameworks de workflow

## Superpowers

Framework de workflow pour agents codeurs, installable via le marketplace Claude Code. Propose un processus structuré en 7 étapes : brainstorming → worktrees → plan → sous-agents parallèles → TDD → révision → finalisation.

Ce qui le distingue : les plans sont décomposés en **tâches de 2-5 minutes** avant toute implémentation — une granularité qui rend chaque étape vérifiable et limite les dérives.

**Installation :**
```
/plugin marketplace add obra/superpowers
```
