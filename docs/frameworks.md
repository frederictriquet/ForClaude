# Frameworks de workflow

## Superpowers

Framework de workflow pour agents codeurs, installable via le marketplace Claude Code. Propose un processus structuré en 7 étapes : brainstorming → worktrees → plan → sous-agents parallèles → TDD → révision → finalisation.

Ce qui le distingue : les plans sont décomposés en **tâches de 2-5 minutes** avant toute implémentation — une granularité qui rend chaque étape vérifiable et limite les dérives.

**Installation :**
```
/plugin marketplace add obra/superpowers
```

→ [GitHub obra/superpowers](https://github.com/obra/superpowers)

## BMAD Method

Framework agile piloté par l'IA, centré sur la collaboration humain-machine plutôt que l'autonomie totale. Propose 12+ personas d'agents spécialisés (PM, architecte, dev, UX, scrum master…) et 34+ workflows structurés couvrant l'intégralité du cycle de développement.

Ce qui le distingue : le **Party Mode** permet à plusieurs personas de collaborer dans une même session, et la complexité de planification s'adapte automatiquement à l'échelle du projet (bugfix → système enterprise).

**Installation :**
```
npx bmad-method install
```

→ [GitHub bmad-code-org/BMAD-METHOD](https://github.com/bmad-code-org/BMAD-METHOD)
