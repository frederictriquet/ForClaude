# Workflow Frameworks

## Superpowers

Workflow framework for coding agents, installable via the Claude Code marketplace. Proposes a structured 7-step process: brainstorming → worktrees → plan → parallel sub-agents → TDD → review → finalization.

What sets it apart: plans are broken down into **2-5 minute tasks** before any implementation — a level of granularity that makes each step verifiable and limits scope creep.

**Installation:**
```
/plugin marketplace add obra/superpowers
```
